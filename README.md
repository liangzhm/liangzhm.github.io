## 博客迁移到mac电脑，每次更新完博客后，需要sudo hexo d 进行部署
### hexo g  生成静态文件
### 命令备份
    hexo g
    sudo hexo d  部署
    hexo s 本地启动
    hexo new 文章名字  新建文章
    sudo ssh -T git@github.com 检测是否链接到了github
